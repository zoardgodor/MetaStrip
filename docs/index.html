MetaStrip

Privacy-focused metadata editor for images, audio, and document files

Mi az a MetaStrip?

A MetaStrip egy offline, adatvédelmi fókuszú Windows alkalmazás, amely lehetővé teszi képek, dokumentumok és médiafájlok metaadatainak megtekintését, szerkesztését és eltávolítását.

Főbb funkciók

  • Metaadatok megtekintése: EXIF, XMP, ID3, PDF
  • Metaadatok szerkesztése: módosítás, új bejegyzés hozzáadása, törlés
  • Metaadatok eltávolítása teljesen
  • Kijelölt metaadatok másolása (Ctrl+C)
  • Fájl tulajdonságok megjelenítése (méret, dátumok, DPI, képméret, megapixel)
  • Többnyelvű felület (magyar, angol)
  • Támogatott fájlok: JPEG, PNG, TIFF, BMP, GIF, WebP, ICO, SVG, MP3, FLAC, OGG, WAV, PDF, DOCX, PPTX, és sok más
  • Felhasználóbarát, fülekre szervezett PySide6 felület
  • Offline működés – teljes adatvédelem

What is MetaStrip?

MetaStrip is an offline, privacy-focused Windows application for viewing, editing, and removing metadata from images, documents, and media files.

Main Features

  • View Metadata: EXIF, XMP, ID3, PDF
  • Edit Metadata: modify, add new entries, delete entries
  • Remove Metadata completely
  • Copy selected metadata (Ctrl+C)
  • File properties display: size, dates, DPI, dimensions, megapixels
  • Multilingual interface (English, Hungarian)
  • Supported files: JPEG, PNG, TIFF, BMP, GIF, WebP, ICO, SVG, MP3, FLAC, OGG, WAV, PDF, DOCX, PPTX ...etc.
  • User-friendly PySide6 tabbed interface
  • Offline operation – full privacy protection

Usage

  • Open File: click "Open File" button
  • View metadata in separate tabs (EXIF, File Properties, Audio Metadata, PDF Metadata)
  • Copy: right click or Ctrl+C
  • Edit Metadata: "Edit" button – modify, add new, delete, save changes
  • Remove Metadata: "Remove Metadata" button
  • Language Switch: "Language" menu – selection stored automatically
